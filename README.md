# NA-SAYINGS PROJECT
## Progetto-Tecnologie-Web

Progetto in fase di sviluppo per l'esame universitario in "Tecnologie Web" de "L'Orientale", primo anno magistrale.

**Per Lara**
* Per entrare nel progetto e fare in modo che anche tu possa collaborare, devi aprire il terminale (o su PHP o uno qualsiasi) e copiare l'indirizzo del mio repository Github, che sarebbe:

git clone https://github.com/mariagiannelavigna/Na-sayings.git

* Dopo aver clonato, nel terminale devi scrivere:

cd Na-sayings

cd sta per "change directory" ed entri nella cartella dove ci sono i file.

* Spero che tu abbia Docker scaricato altrimenti installalo, perch√© dopo, devi aprire Docker e, stesso nel terminale su PHP devi scrivere:

docker compose up -d # dice a Docker di creare e avviare i container

* Infine, vai su Chrome o qualsiasi browser e scrivi:

http://localhost:8000
